# Cinesearchpro - Backend
1. Clonar repositorio
2. Copiar el ```env.template``` y  renombrar a ```.env```
3. Instalar librerias
```
yarn install
```
4. Levantar la imagen de bbdd  (Docker desktop)
```
docker-compose up -d
```
5. Levantar backend de Nest
```
yarn start:dev
```
6. Visitar admin
```
localhost:3000/graphql
```
